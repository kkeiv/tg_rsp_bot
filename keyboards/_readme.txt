Пакет с модулями, в которых хранятся и/или динамически формируются клавиатуры, отправляемые пользователям ботом, в процессе взаимодействия.

__init__.py	- инициализатора пакета

keyboard_utils.py	- Вспомогательные функции/методы, помогающие формировать клавиатуры.

set_menu.py			- Модуль для установки команд в нативную кнопку "Menu" вашего бота.
